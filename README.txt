Для начала работы с программой зайдите в папку "MySQL" и запустите файл "MySQL_start.exe". В этом файле уже хранятся данные по трем основным разделам: "Пределы", "Производные" и "Интегралы".

По окончании работы не забудьте закрыть сервер, запустив файл "MySQL_shutdown.exe".
